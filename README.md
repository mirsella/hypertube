# hypertube

Movie streaming app built with Nuxt, PostgreSQL, subtitles, and torrent-backed media fetching.

## Run locally

```bash
cp .env.example .env
pnpm install
pnpm db:push
pnpm dev
```

## Full stack

```bash
docker compose up --build
```
